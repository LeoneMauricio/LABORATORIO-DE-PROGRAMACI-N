- Qué es Git y para qué sirve? 
Git es una herramienta de control de versiones. Sirve principalmente para guardar, organizar y controlar los cambios que haces en archivos, especialmente en proyectos de programación.

- Los 5 comandos que más vas a usar
1) git init
2) git add .
3) git commit -m "primer commit"
4) git status
5) git push origin main
